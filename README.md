# Personal Portfolio & Guestbook

**Deployed URL:** [Live Website](http://209.38.128.251:3010/)

## Project Description

A full-stack web application built for SDEV 305. Features a professional portfolio, a database-backed guestbook, and a secure admin dashboard.

## Technologies Used

- **Frontend:** EJS, CSS3 (Responsive Design)
- **Backend:** Node.js, Express
- **Database:** MySQL
- **Deployment:** DigitalOcean Droplet, PM2, Nginx

## Setup

1. Clone the repo
2. Run npm install
3. Create a .env file with DB_HOST, DB_USER, DB_PASS, ADMIN_USER, ADMIN_PASS, and SESSION_SECRET.
4. Start with node app.js or pm2 start app.js.
